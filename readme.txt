The Z-Score Calculator helps to get the z-score of a normal distribution, convert between z-score and probability, and get the probability between 2 z-scores.

[https://www.calculator.io/z-score-calculator/](https://www.calculator.io/z-score-calculator/)

== Usage ==

Add the Z-Score Calculator shortcode to your page, post or sidebar:

`[ci_z_score_calculator]`

Add the following code to your template where you would like the Z-Score Calculator to appear:

`<?php display_ci_z_score_calculator(); ?>`

== Installation ==

1. Upload the Z-Score Calculator /ci_z_score_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Z-Score Calculator plugin through the "Plugins" menu in WordPress.
